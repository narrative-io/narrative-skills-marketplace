# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1][cc].

In short: be respectful, assume good intent, focus on the work, and
make this a welcoming space for contributors of any background. Harassment,
discrimination, or personal attacks are not tolerated.

## Reporting

If you experience or witness conduct that violates this code, please
contact the maintainers at <dev@narrative.io>. Reports will be
reviewed promptly and confidentially.

The full text and enforcement guidelines live at:

- <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>
- <https://www.contributor-covenant.org/version/2/1/code_of_conduct/code_of_conduct.md>
